# API Documentation

## Swagger/OpenAPI

Para acessar a documentação interativa da API via Swagger:

1. **Inicie o servidor da API**
2. **Acesse o Swagger UI no navegador:**
   ```
   http://localhost:3000/api-docs
   ```

### Comandos úteis:

```bash
# Iniciar o servidor em modo desenvolvimento
npm run dev

# Iniciar o servidor em modo produção
npm start
```

A documentação do Swagger estará disponível em `http://localhost:3000/api-docs` após o servidor estar executando.
